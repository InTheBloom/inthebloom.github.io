гҒҜгҒҳгӮҒгҒ«

AWC0008D - жһңжЁ№ең’гҒ®еҸҺз©« гӮ’$O(N(\log N + \log (\max F _ i)))$жҷӮй–“гҒ§и§ЈгҒҚгҒҫгҒҷгҖӮиҰҒгҒҜ$M \leq 10 ^ 9$гҒ§гӮӮи§ЈгҒ‘гӮӢи§Јжі•гҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҒҷгҖӮ и§ЈиӘ¬гҒ§иЁҖеҸҠгҒ•гӮҢгҒҰгҒ„гӮӢгҒ®гҒ§ж–°иҰҸжҖ§гҒҜгҒӘгҒ„гҒ§гҒҷгҖӮ

е•ҸйЎҢжҰӮиҰҒ

$N$жң¬гҒ®жңЁгҒҢгҒӮгӮӢгҖӮжңЁ$i$гҒҜ$k$еӣһзӣ®гҒ®еҸҺз©«гҒ§гҒҜ$\max(F _ i - (k - 1) \times D _ i, 0)$еҖӢгҒ®жһңзү©гҒҢеҸ–гӮҢгӮӢгҖӮ $M$еӣһд»ҘдёӢгҒ®еҸҺз©«гҒ§еҫ—гӮүгӮҢгӮӢжһңзү©гҒ®жңҖеӨ§еҖӨгӮ’жұӮгӮҒгӮҲгҖӮ

  • $1 \leq N \leq 2 \times 10 ^ 5$
  • $1 \leq M \leq 2 \times 10 ^ 5$
  • $1 \leq F _ i \leq 10 ^ 9$
  • $1 \leq D _ i \leq 10 ^ 9$

O((M + N)logN)жҷӮй–“и§Јжі•

жңЁ$i$гҒ®еҸҺз©«йҮҸгҒ®еӨүеҢ–гҒҜд»–гҒ®жңЁгҒЁзӢ¬з«ӢгҒӘгҒ®гҒ§гҖҒд»ҠеҸҺз©«йҮҸгҒҢжңҖеӨ§гҒ®гӮӮгҒ®гӮ’йҒёгҒіз¶ҡгҒ‘гӮӢгҒЁгҒ„гҒҶиІӘж¬ІгҒҢжӯЈеҪ“жҖ§гӮ’жҢҒгҒЎгҒҫгҒҷгҖӮ иЁјжҳҺгҒЁгҒ—гҒҰгҒҜгҖҒгҖҢеҸҺз©«йҮҸгҒҢжңҖеӨ§гҒ®гӮӮгҒ®гӮ’йҒёгҒ°гҒӘгҒ„жҷӮгҒҢгҒӮгӮҢгҒ°гҖҒжңҖеӨ§гҒ®гӮӮгҒ®гӮ’йҒёгҒ¶гӮҲгҒҶгҒ«гҒҷгӮӢгҒ“гҒЁгҒ§гӮ№гӮігӮўгӮ’ж”№е–„гҒҷгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгӮӢгҖҚгҒЁгҒ„гҒҶгҒ“гҒЁгӮ’еҺіеҜҶгҒ«жӣёгҒ‘гҒ°гӮҲгҒ„гҒҜгҒҡгҒ§гҒҷгҖӮ

гӮҲгҒЈгҒҰгҖҒжңҖеӨ§гғ’гғјгғ—гҒ«$(F _ i, i)$гӮ’е…ҘгӮҢгҒҰгҖҒ

  1. жңҖеӨ§гӮ’гҒЁгҒЈгҒҰи¶ігҒҷ
  2. $D _ i$гӮ’еј•гҒҸ
  3. гғ’гғјгғ—гҒ«жҲ»гҒҷ

гҒЁгҒ„гҒҶгҒ“гҒЁгӮ’$M$еӣһгӮ„гӮӢгҒЁи§ЈгҒ‘гҒҫгҒҷгҖӮ

O(N(logN + logF))жҷӮй–“и§Јжі•

жңҖеӨ§гҒ®гӮӮгҒ®гӮ’йҒёгҒіз¶ҡгҒ‘гӮҢгҒ°гӮҲгҒ„гҒЁгҒ„гҒҶгҒ“гҒЁгҒҜгҖҒ$M$еӣһгҒ®ж“ҚдҪңгҒҢзөӮгӮҸгҒЈгҒҹеҫҢгҖҒгҒЁгҒігҒ¬гҒ‘гҒҰеӨ§гҒҚгҒ„еҖӨгҒҜеҸҺз©«гҒ«гӮҲгҒЈгҒҰеқҮгҒ•гӮҢгҒҰгҖҒеҸҺз©«йҮҸгҒҜгҒ©гӮҢгӮӮгҒӮгӮӢзЁӢеәҰеҗҢгҒҳеҖӨгҒ«гҒӘгҒЈгҒҰгҒ„гӮӢгҒҜгҒҡгҒ§гҒҷгҖӮпјҲе…ғгҒ®еҸҺз©«йҮҸгҒҢйқһеёёгҒ«е°ҸгҒ•гҒӢгҒЈгҒҹгӮҠгҖҒ$D$гҒҢйқһеёёгҒ«еӨ§гҒҚгҒӢгҒЈгҒҹгӮҠгҒҷгӮӢгҒЁгҒқгҒ“гҒ гҒ‘еҮ№гӮ“гҒ гӮҲгҒҶгҒ«гҒӘгӮҠгҒҫгҒҷгҒҢгҖӮпјү гҒқгҒ“гҒ§гҖҒеҸҺз©«зөӮдәҶеҫҢгҒ®еҸҺз©«йҮҸгҒ®жңҖеӨ§еҖӨгҒҢгҒ„гҒҸгҒӨгҒ«гҒӘгӮӢгҒӢгӮ’иҖғгҒҲгҒҰгҒҝгҒҫгҒҷгҖӮ

дәҢеҲҶжҺўзҙўгӮ’иЎҢгҒ„гҒҫгҒҷгҖӮ еҸҺз©«йҮҸгҒ®жңҖеӨ§еҖӨгҒҢ$x$жңӘжәҖгҒ«гҒӘгӮӢгҒЁжұәгӮҒжү“гҒЎгҒ—гҒҹйҡӣгҖҒеҝ…иҰҒгҒӘеҸҺз©«еӣһж•°гҒ®жңҖе°ҸеҖӨгҒҢгӮҸгҒӢгӮҠгҒҫгҒҷгҒӢгӮүгҖҒгҒ“гӮҢгҒҢ$M$д»ҘдёӢгҒ«гҒӘгӮӢжңҖеӨ§еҖӨгӮ’жҺўгҒҷгҒ“гҒЁгҒҢгҒ§гҒҚгҒҫгҒҷгҖӮ гҒ“гӮҢгҒҢгӮҸгҒӢгӮҢгҒ°гҖҒеҸҺз©«йҮҸгҒҢеӨ§дҪ“гҒқгҒ®еҖӨгҒ«гҒӘгӮӢгҒҫгҒ§гҒҜдёҖж°—гҒ«еҸҺз©«гӮ’йҖІгӮҒгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгӮӢгҒҹгӮҒгҖҒе…ғгҒ®и§Јжі•гҒ®гӮҲгҒҶгҒ«$M$еӣһе®ҹйҡӣгҒ«ж“ҚдҪңгҒҷгӮӢеҝ…иҰҒгҒҢгҒӘгҒҸгҒӘгӮҠгҒҫгҒҷгҖӮ

гҒ•гҒҰгҖҒ$M$еӣһгҒ®ж“ҚдҪңгӮ’гҒ“гҒ®еӨ§дҪ“гҒ®зӣ®е®үгҒ§ж¶ҲиІ»гҒ—гҒҚгӮҢгҒӘгҒ„е ҙеҗҲгҒҢгҒӮгӮҠгҒҫгҒҷгҒҢгҖҒгҒқгҒ®гӮҲгҒҶгҒӘе ҙеҗҲгҖҒе®ҹгҒҜдҪҷгҒЈгҒҹж“ҚдҪңеӣһж•°гҒҜ$N$еӣһд»ҘдёӢгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

гҒ“гӮҢгҒҜгҖҒ$f(x) = еҸҺз©«йҮҸгҒҢxд»ҘдёҠгҒ®гӮӮгҒ®гҒҜеҸҺз©«гҒҷгӮӢгҒЁгҒҚгҖҒеҸҺз©«еӣһж•°гҒ®жңҖе°ҸеҖӨгҒҜдҪ•еӣһгҒӢ$гҒЁгҒ—гҒҹгҒЁгҒҚгҖҒ$f(x + 1)$гҒЁ$f(x)$гҒ®е·®гӮ’иҖғгҒҲгӮӢгҒЁгӮҸгҒӢгӮҠгҒҫгҒҷгҖӮ $f(x + 1)$гӮҲгӮҠ$f(x)$гҒҢ$N + 1$д»ҘдёҠеӨҡгҒ„гҒЁгҒҚгҖҒйі©гҒ®е·ЈеҺҹзҗҶгҒӢгӮүеҸҺз©«еӣһж•°гҒҢ2еӣһд»ҘдёҠеў—гҒҲгҒҹжңЁгҒҢеӯҳеңЁгҒ—гҒҫгҒҷгҖӮ гҒ—гҒӢгҒ—$f$гҒ®е®ҡзҫ©гӮ’иҖғгҒҲгӮӢгҒЁгҖҒ$f(x + 1)$гҒ®жҷӮзӮ№гҒ§е…ЁгҒҰгҒ®жңЁгҒ®еҸҺз©«йҮҸгҒҜ$x$д»ҘдёӢгҒ«гҒӘгҒЈгҒҰгҒ„гӮӢгҒҜгҒҡгҒ§гҒҷгҖӮ $1 \leq D _ i$гҒ§гҒӮгӮӢгҒ“гҒЁгҒӢгӮүгҖҒ1еӣһгҒ®еҸҺз©«гҒ§еҝ…гҒҡ$x$жңӘжәҖгӮ’йҒ”жҲҗгҒ§гҒҚгӮӢгҒҹгӮҒгҖҒ2еӣһд»ҘдёҠеҸҺз©«гҒ—гҒҹгҒЁгҒ„гҒҶгҒ®гҒҜгҒӮгӮҠгҒҲгҒҫгҒӣгӮ“гҖӮ $f$гҒ®еҖӨгҒҜ$N$д»ҘдёӢгҒ®еӨүеҢ–гҒ—гҒӢгҒ—гҒӘгҒ„гҒҹгӮҒгҖҒдҪҷгҒЈгҒҹж“ҚдҪңеӣһж•°гӮӮеҝ…гҒҡ$N$д»ҘдёӢгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

гҒ“гҒ®2гҒӨгӮ’зө„гҒҝеҗҲгӮҸгҒӣгҒҰгҖҒж¬ЎгҒ®и§Јжі•гӮ’еҫ—гҒҫгҒҷгҖӮ

  1. дәҢеҲҶжҺўзҙўгҒ«гӮҲгӮҠгҖҒ$M$еӣһгҒ®еҸҺз©«гӮ’жңҖйҒ©гҒ«иЎҢгҒЈгҒҹеҫҢгҒ®жңҖеӨ§еҸҺз©«йҮҸ$x$гӮ’жұӮгӮҒгӮӢ
  2. еҸҺз©«йҮҸгҒҢ$x$д»ҘдёӢгҒ«гҒӘгӮӢгҒҫгҒ§гҒҜдёҖж°—гҒ«еҸҺз©«гӮ’иЎҢгҒҶ
  3. дҪҷгҒЈгҒҹеҸҺз©«еӣһж•°гӮ’жңҖеӨ§гғ’гғјгғ—гҒ§ж¶ҲеҢ–гҒҷгӮӢ

жҷӮй–“иЁҲз®—йҮҸгҒҜдәҢеҲҶжҺўзҙўгҒҢ$O(N \log (\max F _ i))$гҒ§гғ’гғјгғ—гӮ’з”ЁгҒ„гӮӢйғЁеҲҶгҒҢ$O(N \log N)$гҒ§гҒҷгҖӮ

е®ҹиЈ…

гҒ“гҒ®жүӢгҒ®дәҢеҲҶжҺўзҙўгҒЁгҒ—гҒҰгҒҜе®ҹиЈ…гҒҢгӮ„гӮ„йӣЈгҒ—гҒ„гҒ§гҒҷгҖӮ ж°—гӮ’д»ҳгҒ‘гӮӢгҒ№гҒҚгғқгӮӨгғігғҲгҒҜгҒ„гҒҸгҒӨгҒӢгҒӮгӮҠгҒҫгҒҷгҖӮ

  1. дәҢеҲҶжҺўзҙўгҒ®еҲӨе®ҡе•ҸйЎҢгҒ®з«ӢгҒҰж–№гҒ«гӮҲгҒЈгҒҰloеҒҙгҒӢhiеҒҙгҒ®еҖӨгҒҢеӯҳеңЁгҒ—гҒӘгҒҸгҒӘгӮӢгҒ“гҒЁгҒҢгҒӮгӮӢ
  2. жң¬жқҘеҸҺз©«йҮҸгҒҢ0жңӘжәҖгҒ«гҒӘгӮүгҒӘгҒ„гҒҢгҖҒеҲӨе®ҡе•ҸйЎҢгӮ’и§ЈгҒҸйҡӣгҒ«гҒҜеҸҺз©«йҮҸгҒ«иІ гӮ’иЁұе®№гҒ—гҒҹгҒ»гҒҶгҒҢгӮҲгҒ„пјҲгҖҢ$x$д»ҘдёҠгҒҜеҸҺз©«гҒҷгӮӢгҖҚгҒЁгҒ„гҒҶеҲӨе®ҡе•ҸйЎҢгҒ гҒЁ0гҒӢгӮүдёӢгҒҢгӮүгҒӘгҒҸгҒӘгӮӢгҒ®гҒҜйқўеҖ’пјү
  3. дёҖж°—гҒ«еҸҺз©«гӮ’иЎҢгҒҶйҡӣгҖҒеҹәжң¬зҡ„гҒ«гҒҜзӯүе·®ж•°еҲ—гҒ®з·Ҹе’ҢгӮ’иҖғгҒҲгӮҢгҒ°иүҜгҒ„гҒҢгҖҒжңҖеҫҢгҒ®1еӣһгҒҜеҸҺз©«йҮҸгҒҢ0гҒ«гҒӘгӮӢеҲҶгӮ’иҖғж…®гҒ—гҒӘгҒ„гҒЁгҒ„гҒ‘гҒӘгҒ„гӮұгғјгӮ№гҒҢгҒӮгӮӢгҒ®гҒ§гҖҒжңҖеҫҢгҒ®ж•°еӣһгҒҜwhileгҒЁгҒӢгҒ§гӮ„гӮӢгҒЁе®үе…Ё

гҒЁгҒ„гҒЈгҒҹж„ҹгҒҳгҒ§гҒҷгҖӮ з§ҒгҒҜ$O(1)$з®—ж•°йғЁеҲҶгҒ§иӢҰжҲҰгҒ—гҒҰ1жҷӮй–“гҒҸгӮүгҒ„гҒӢгҒӢгӮҠгҒҫгҒ—гҒҹгҖӮ з§ҒгҒ®еҲӨе®ҡе•ҸйЎҢгҒҜ$f(x) = xд»ҘдёҠгҒЁгӮҢгӮӢжңЁгӮ’е…ЁйғЁеҸ–гӮӢгҒЁгҒ—гҒҰгҖҒеҸҺз©«еӣһж•°гҒҢMд»ҘдёӢгҒ«гҒӘгӮӢгҒӢ$гҒ гҒЈгҒҹгҒ®гҒ§гҒҷгҒҢгҖҒгҒ“гҒ®йҡӣгҒӮгӮӢжңЁгҒ«еҝ…иҰҒгҒӘеҸҺз©«еӣһж•°гӮ’ $$ F _ i - k \times D _ i < x $$ гӮ’жәҖгҒҹгҒҷжңҖе°ҸгҒ®$k$гҒЁгҒ—гҒҰиҖғгҒҲгҒҫгҒ—гҒҹгҖӮгҒ“гӮҢгӮ’$k$гҒ«гҒӨгҒ„гҒҰи§ЈгҒҸгҒЁгҖҒ $$ \frac{F _ i - x}{D _ i} < k $$ гҒЁгҒӘгӮҠгҒҫгҒҷгҖӮгҒ“гҒ“гҒҫгҒ§гҒҜиүҜгҒ„гҒ®гҒ§гҒҷгҒҢгҖҒе®ҹиЈ…дёҠгҒҜ$F _ i - x < 0$гҒӢгҒ©гҒҶгҒӢгҒ«ж°—гӮ’й…ҚгӮүгҒӘгҒ„гҒЁдәӢж•…гӮҠгҒҫгҒҷгҖӮ $F _ i - x < 0$гҒ§гҒӮгӮҢгҒ°$k = 0$гҒ§гӮҲгҒҸгҒҰгҖҒгҒқгҒҶгҒ§гҒӘгҒ„е ҙеҗҲгҒҜ$k$гҒҢж•ҙж•°гҒӘгҒ“гҒЁгҒӢгӮү $$ \left\lfloor \frac{F _ i - x}{D _ i} \right\rfloor + 1 \leq k $$ гҒЁгҒ§гҒҚгҒҫгҒҷгҖӮ

ж®ӢгӮҠгҒ®жіЁж„ҸзӮ№гҒҜзӯүе·®ж•°еҲ—гҒ®з·Ҹе’ҢгҒҸгӮүгҒ„гҒ§гҒҷгҒҢгҖҒеҲ¶зҙ„дёҠlongгҒӘгӮүгҒҫгҒӮдҪҷиЈ•гҒӘгҒ®гҒ§

long calc (long f, long d, long c) {
    // еҲқй …fгҖҒе…¬е·®dгҖҒй …ж•°cгҒ®зӯүе·®ж•°еҲ—гҒ®е’Ң
    long l = f + d * (c - 1);
    return (f + l) * c / 2;
}

гҒЁгҒ„гҒҶж„ҹгҒҳгҒ§гӮ„гӮҠгҒҫгҒ—гҒҹгҖӮ

жңҖеҫҢгҒ«и§Јзӯ”дҫӢгӮ’дёҖйғЁжҠңзІӢгҒ§жҺІијүгҒ—гҒҫгҒҷгҖӮ

import std;

void main () {
    int N, M;
    readln.read(N, M);
    auto F = new int[](N);
    auto D = new int[](N);
    foreach (i; 0 .. N) {
        readln.read(F[i], D[i]);
    }

    long calc (long f, long d, long c) {
        // еҲқй …fгҖҒе…¬е·®dгҖҒй …ж•°cгҒ®зӯүе·®ж•°еҲ—гҒ®е’Ң
        long l = f + d * (c - 1);
        return (f + l) * c / 2;
    }

    bool f (int x) {
        // xеҖӢд»ҘдёҠеҸ–гӮҢгӮӢжңЁгӮ’е…ЁйғЁеҸ–гӮӢгҒЁгҒ—гҒҰгҖҒеҸҺз©«еӣһж•°гҒҢMд»ҘдёӢгҒ«гҒӘгӮӢгҒӢпјҹ
        long took = 0;
        foreach (i; 0 .. N) {
            // F[i] - k * D[i] < x
            // гҒЁгҒӘгӮӢжңҖе°ҸгҒ®k
            // F[i] - x < k * D[i]
            // (F[i] - x) / D[i] < k

            long v = (1L * F[i] - x) / D[i] + 1;
            v = max(0L, v);

            if (F[i] - x < 0) {
                v = 0;
            }
            took += v;
        }

        return took <= M;
    }
    auto ret = bsearch!(f)(10 ^^ 9 + 10, 0);

    long ans = 0;
    long use = 0;
    // иІ гҒ«гҒӘгӮӢгҒ“гҒЁгӮ’иЁұе®№гҒ—гҒҰгҖҒF[i] < ret.valueгҒЁгҒӘгӮӢгҒҫгҒ§еҸ–гӮӢ
    foreach (i; 0 .. N) {
        long v = (1L * F[i] - ret.value) / D[i] + 1;
        if (F[i] - ret.value < 0) {
            v = 0;
        }

        // гҒЎгӮҮгҒЈгҒЁгғҗгғғгғ•гӮЎгӮ’жҢҒгҒҹгҒӣгӮӢ
        v = max(0L, v - 5);

        ans += calc(F[i], -D[i], v);
        F[i] -= v * D[i];
        use += v;

        // з«Ҝж•°
        while (ret.value <= F[i]) {
            ans += F[i];
            F[i] -= D[i];
            use++;
        }
    }

    // ж®ӢгӮҠгҒҜpqгҒ§иӘҝж•ҙ
    auto pq = BinaryHeap!(Tuple!(long, int)[], "a[0] < b[0]")([]);
    foreach (i; 0 .. N) {
        pq.insert(tuple(1L * F[i], i));
    }

    while (use < M) {
        auto v = pq.front();
        pq.removeFront();
        use++;
        if (0 <= v[0]) {
            ans += v[0];
        }
        v[0] -= D[v[1]];
        pq.insert(v);
    }

    writeln(ans);
}
