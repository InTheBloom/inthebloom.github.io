ж”№з«„гҒ•гӮҢгҒҹеҸІж–ҷ

еӨӘеҸӨгҒ®жҳ”пјҢ International Collegiate Puzzle Contest (ICPC) гҒЁгҒ„гҒҶгғ‘гӮәгғ«еӨ§дјҡгҒҢжҜҺе№ҙй–ӢеӮ¬гҒ•гӮҢгҒҰгҒ„гҒҹпјҺ еҗ„еӨ§дјҡгҒҜеҚҲеүҚгғ»еҚҲеҫҢгҒ® 2 гҒӨгҒ®гғ©гӮҰгғігғүгҒ§ж§ӢжҲҗгҒ•гӮҢгҒҰгҒ„гҒҹпјҺ гҒ©гҒЎгӮүгҒ®гғ©гӮҰгғігғүгҒ§гӮӮпјҢеҸӮеҠ иҖ…гҒҜгҒҫгҒҡиҮӘдҪңгҒ®гғ‘гӮәгғ« 1 гҒӨгӮ’д»–гҒ®е…ЁеҸӮеҠ иҖ…гҒ«жҸҗзӨәгҒ—пјҢгҒқгҒ®еҫҢгҒ«иҮӘеҲҶиҮӘиә«д»ҘеӨ–гҒ®е…ЁеҸӮеҠ иҖ…гҒҢжҸҗзӨәгҒ—гҒҹгғ‘гӮәгғ«гӮ’гҒ§гҒҚгӮӢгҒ гҒ‘еӨҡгҒҸи§ЈгҒ“гҒҶгҒЁгҒ—гҒҹпјҺ

еҸӮеҠ иҖ…гҒ®й ҶдҪҚгҒҜд»ҘдёӢгҒ®иҰҸеүҮгҒ«еҫ“гҒЈгҒҰжұәгӮҒгҒҰгҒ„гҒҹпјҺ

  • еҚҲеҫҢгҒ®гғ©гӮҰгғігғүгҒ§и§ЈгҒ„гҒҹгғ‘гӮәгғ«гҒ®ж•°гҒҢгӮҲгӮҠеӨҡгҒ„еҸӮеҠ иҖ…гҒҢгӮҲгӮҠдёҠдҪҚгҒЁгҒӘгӮӢпјҺ
  • еҚҲеҫҢгҒ®гғ©гӮҰгғігғүгҒ§еҗҢж•°гҒ®гғ‘гӮәгғ«гӮ’и§ЈгҒ„гҒҹеҸӮеҠ иҖ…гҒ®гҒӘгҒӢгҒ§гҒҜпјҢеҚҲеүҚгҒ®гғ©гӮҰгғігғүгҒ§и§ЈгҒ„гҒҹгғ‘гӮәгғ«гҒ®ж•°гҒҢгӮҲгӮҠеӨҡгҒ„ж–№гҒҢдёҠдҪҚгҒЁгҒӘгӮӢпјҺ
  • дёЎгғ©гӮҰгғігғүгҒ§и§ЈгҒ„гҒҹгғ‘гӮәгғ«гҒ®ж•°гҒҢе®Ңе…ЁгҒ«дёҖиҮҙгҒҷгӮӢеҸӮеҠ иҖ…еҗҢеЈ«гҒ®й ҶдҪҚгҒҜгҒҸгҒҳгҒігҒҚгҒ§жұәгӮҒгӮӢпјҺ

е…Ҳж—ҘпјҢгҒӮгӮӢе№ҙгҒ® ICPC гҒ®й ҶдҪҚиЎЁгҒЁгҒ•гӮҢгӮӢж–ҮжӣёгҒҢзҷәиҰӢгҒ•гӮҢгҒҹпјҺ д»–гҒ®еҗҢж§ҳгҒ®иіҮж–ҷгҒӢгӮүпјҢгҒ“гҒ®ж–ҮжӣёгҒҜеҗ„гғ©гӮҰгғігғүгҒ§и§ЈгҒ„гҒҹгғ‘гӮәгғ«гҒ®ж•°гӮ’дёҠдҪҚгҒ®еҸӮеҠ иҖ…гҒӢгӮүй ҶгҒ«жӣёгҒҚдёҰгҒ№гҒҹгӮӮгҒ®гҒЁиҖғгҒҲгӮүгӮҢгӮӢпјҺ гҒЁгҒ“гӮҚгҒҢеӯҰиҖ…гҒҹгҒЎгҒҜж–ҮжӣёгҒҜдҪ•иҖ…гҒӢгҒ«гӮҲгҒЈгҒҰж”№з«„гҒ•гӮҢгҒҰгҒ„гҒҰпјҢгҒ„гҒҸгҒӨгҒӢгҒ®ж•°гҒҢжӣёгҒҚжҸӣгҒҲгӮүгӮҢгҒҰгҒ„гӮӢгҒ®гҒ§гҒҜгҒӘгҒ„гҒӢгҒЁгҒ„гҒҶз–‘гҒ„гӮ’жҢҒгҒЈгҒҰгҒ„гӮӢпјҺ гӮӮгҒЁгӮӮгҒЁгҒ®й ҶдҪҚиЎЁгҒ«дёҠиҝ°гҒ®иҰҸеүҮгҒЁзҹӣзӣҫгҒҢгҒӘгҒӢгҒЈгҒҹгҒЁд»®е®ҡгҒ—гҒҹгҒЁгҒҚпјҢе°‘гҒӘгҒҸгҒЁгӮӮгҒ„гҒҸгҒӨгҒ®ж•°гҒҢжӣёгҒҚжҸӣгҒҲгӮүгӮҢгҒҰгҒ„гӮӢгҒ®гҒӢгӮ’жұӮгӮҒгҒҰгҒ»гҒ—гҒ„пјҺ гҒӘгҒҠпјҢж–ҮжӣёдёӯгҒ®и§ЈгҒ„гҒҹгғ‘гӮәгғ«гҒ®ж•°д»ҘеӨ–гҒ®йғЁеҲҶгҒёгҒ®ж”№з«„гҒҜгҒӘгҒҸпјҢеҸӮеҠ иҖ…гҒ®иҝҪеҠ гӮ„еүҠйҷӨгӮӮгҒӮгӮҠгҒҲгҒӘгҒ„пјҺ

ICPCгҒ®еҸӮеҠ иҖ…$n$гҒҜгҒІгҒЁгҒӨгҒ®гғҶгӮ№гғҲгӮұгғјгӮ№гҒ§$n \leq 6000$гҒЁгҒӘгӮӢгҖӮеҗ«гҒҫгӮҢгӮӢгғҶгӮ№гғҲгӮұгғјгӮ№гҒҜгҒІгҒЁгҒӨгҒ®гғ•гӮЎгӮӨгғ«гҒ§й«ҳгҖ…100гҒ§гҒӮгӮҠгҖҒ$n$гҒ®еҗҲиЁҲгҒҜ$10 ^ 5$гӮ’и¶…гҒҲгҒӘгҒ„гҖӮ

е•ҸйЎҢгҒ®иЁҖгҒ„гҒӢгҒҲ

е•ҸйЎҢгӮ’з«¶гғ—гғӯйўЁгҒ«иЁҖгҒ„жҸӣгҒҲгӮӢгҖӮ $n \leq 6000$еҖӢгҒ®ж•°гҒ®гғҡгӮў$(x _ i, y _ i)$гҒҢдёҺгҒҲгӮүгӮҢгӮӢгҖӮ дәҢгҒӨгҒ®гғҡгӮў$(x _ i, y _ i), (x _ j, y _ j)$гҒ«еҜҫгҒ—гҒҰгҖҒ

  • $y _ i < y _ j$гҒҫгҒҹгҒҜ$y _ i = y _ j \land x _ i < x _ j$гҒ§гҒӮгӮӢе ҙеҗҲгҒ«йҷҗгӮҠ$(x _ i, y _ i) < (x _ j, y _ j)$
  • $y _ i = y _ j$гҒӢгҒӨ$x _ i = x _ j$гҒ§гҒӮгӮӢе ҙеҗҲгҒ«йҷҗгӮҠ$(x _ i, y _ i) = (x _ j, y _ j)$

гҒ§гҒӮгӮӢгҒЁгҒҷгӮӢгҖӮ д»»ж„ҸгҒ«ж•°гӮ’жӣёгҒҚжҸӣгҒҲгӮӢгҒ“гҒЁгҒ§ж¬ЎгҒ®жқЎд»¶гҒҢжҲҗз«ӢгҒҷгӮӢгӮҲгҒҶгҒ«гҒҷгӮӢгҖӮ

  • е…ЁгҒҰгҒ®ж•°гҒҢ$[0, n)$гҒ®зҜ„еӣІгҒ«гҒӮгӮӢгҖӮ
  • $1 \leq i \leq n - 1$гҒ«еҜҫгҒ—гҒҰгҖҒ$(x _ i, y _ i) \leq (x _ {i + 1}, y _ {i + 1})$

жӣёгҒҚжҸӣгҒҲгӮӢеҝ…иҰҒгҒ®гҒӮгӮӢж•°гҒ®жңҖе°ҸеҖӨгҒҜгҒ„гҒҸгҒӨгҒӢпјҹ

иҖғеҜҹ

  • еҚҲеҫҢгҒ®йғЁгҒ®жҲҗзёҫгҒҢгӮҲгӮҠеј·гҒҸеҠ№гҒҸгҒ®гҒ§гҖҒгҒ“гҒ“гӮ’иүҜгҒ„ж„ҹгҒҳгҒ«жҸғгҒҲгӮӢгҒ“гҒЁгӮ’иҖғгҒҲгҒҰгҒҝгӮӢгҖӮгҒҷгӮӢгҒЁгҖҒ$i$з•Әзӣ®гҒ®гғҡгӮўгҒ®еҚҲеҫҢгҒ®жҲҗзёҫгӮ’$n - i$гҒ«гҒҷгӮӢгҒ“гҒЁгҒ§$n$еӣһгҒ§йҒ”жҲҗгҒ§гҒҚгӮӢгҒ“гҒЁгҒҢгӮҸгҒӢгӮӢгҖӮи§ЈгҒҢ$n$гӮ’и¶…гҒҲгӮӢгҒ“гҒЁгҒҜгҒӘгҒ•гҒқгҒҶгҖӮ

  • йҒ”жҲҗгҒ§гҒҚгӮӢж”№еӨүж•°гҒ«гҒҜеҚҳиӘҝжҖ§гҒҢгҒӮгӮӢгҖӮгҒӨгҒҫгӮҠгҖҒ$k$еӣһгҒ§йҒ”жҲҗгҒ§гҒҚгҒҹгҒЁгҒҷгӮӢгҒЁгҖҒ$k + 1$еӣһгҒ§гӮӮйҒ”жҲҗгҒ§гҒҚгӮӢгҖӮгҒҳгӮғгҒӮдәҢеҲҶжҺўзҙўгҒҝгҒҹгҒ„гҒӘгҒ®гҒ«иҗҪгҒЁгҒ—иҫјгӮҒгӮӢгҒӢпјҹ -> еҲӨе®ҡе•ҸйЎҢгӮ’иүҜгҒ„ж„ҹгҒҳгҒ«иІӘж¬ІиҖғгҒҲгӮҲгҒҶгҒЁгҒ—гҒҰгӮӮгҒӢгҒӘгӮҠеҺігҒ—гҒқгҒҶгҖӮгҒқгӮӮгҒқгӮӮи§ЈгҒ®дёҠз•ҢгҒҢгҒӢгҒӘгӮҠе°ҸгҒ•гҒ„гҒӢгӮүдәҢеҲҶжҺўзҙўгҒ®гҒҶгҒҫгҒҝгҒҢе°‘гҒӘгҒ„гҖӮ

  • еҚҲеҫҢ -> еҚҲеүҚгҒ®й ҶгҒ«гҒқгӮҚгҒҲгӮӢгҒЁгҒ—гҒҹгӮүпјҹ -> еҚҲеҫҢгҒ®йғЁгҒ®жҲҗзёҫгҒҢеҗҢгҒҳгӮ°гғ«гғјгғ—гҒҜзӢ¬з«ӢгҒ«иҖғгҒҲгҒҰгӮҲгҒ„гҖӮгҒІгҒЁгҒӨгҒ®гӮ°гғ«гғјгғ—гҒҢжұәгҒҫгҒЈгҒҰгҒ„гҒҹгӮүжңҖйҒ©гҒӘжүӢй ҶгҒҜгӮҸгҒӢгӮӢпјҹгҒ„гӮ„гҖҒгҒ“гӮҢгҒ гҒ‘гҒ§гӮӮзөҗж§ӢеҺігҒ—гҒқгҒҶгҖӮгҒ—гҒӢгӮӮгӮ°гғ«гғјгғ—гҒ®жұәгӮҒж–№гҒ§е…ЁгҒҰгҒҢеӨүгӮҸгҒЈгҒҰгҒ—гҒҫгҒҶгҒ—гҖҒгӮ°гғ«гғјгғ—гҒ®еҲҶгҒ‘ж–№гӮ’е…ЁйғЁжҺўзҙўгҒҷгӮӢгҒ®гӮӮеҪ“з„¶з„ЎзҗҶгҖӮ

  • жҺўзҙўгғҷгғјгӮ№гҒ®иІӘж¬ІгҒҜеҺігҒ—гҒқгҒҶпјҹз«ҜгҒӢгӮүжұәгӮҒгҒҰгҒҝгҒҹгӮүгҒ©гҒҶгҒ гӮҚгҒҶгҒӢгҖӮ -> дёҖз•ӘгӮ·гғігғ—гғ«гҒӘdpгҒӘгӮү$\mathrm{dp}[i][j][k]$гҒҢе…Ҳй ӯ$i$й …гҒ®й Ҷз•ӘгӮ’гҒқгӮҚгҒҲгҒҰгҖҒдёҖз•ӘжңҖеҫҢгҒ®гғҡгӮўгҒҢ$(j, k)$гҒЁгҒӘгӮӢжңҖе°ҸгӮігӮ№гғҲгҒЁгҒӘгӮҠгҒқгҒҶгҖӮгҒ“гӮҢгҒӘгӮүиЁҲз®—йҮҸгӮ’з„ЎиҰ–гҒ—гҒҹгӮүи§ЈгҒ‘гӮӢгҖӮеӨҡеҲҶеҪ“гҒҹгӮҠж–№йҮқгҒЈгҒҪгҒ„гҖӮ

  • ж–°гҒ—гҒҸгҖҢжңҖеҫҢгҒ®гғҡгӮўгҖҚгӮ’жұәгӮҒгӮӢгҒЁгҒҚгҖҒгҒІгҒЁгҒӨеүҚгҒ®гғҡгӮўгӮҲгӮҠй Ҷз•ӘгҒҢеӨ§гҒҚгҒ„еҝ…иҰҒгҒҢгҒӮгӮӢгҖӮгҒҫгҒҹгҖҒеҮәжқҘгӮӢгҒ гҒ‘е°ҸгҒ•гҒ„ж–№гҒҢеҪ“з„¶иүҜгҒ„гҖӮдҫӢгҒҲгҒ°$(1, 1)$гҒЁгҒӢгҒӢгӮүеӨүжӣҙ2еӣһгӮ’дҪҝгҒЈгҒҰ$(n - 1, n - 1)$гҒ«гӮҸгҒ–гӮҸгҒ–еӨүжӣҙгҒҷгӮӢж„Ҹе‘ігҒҜе…ЁгҒҸгҒӘгҒ„гҖӮеӨүжӣҙ2еӣһгӮӮдҪҝгҒҶгҒӘгӮүеҪ“з„¶$(1, 1)$гҒ«гҒҷгҒ№гҒҚгҖӮгҒ“гӮҢгӮ’дҪҝгҒЈгҒҰгҒӘгӮ“гҒӢгҒ„гҒ„ж„ҹгҒҳгҒ«гҒ§гҒҚгҒқгҒҶпјҹ

  • гғҠгғғгғ—гӮөгғғгӮҜdpгҒ®ж·»еӯ—гӮ’йҖҶи»ўгҒ•гҒӣгӮӢгғҶгӮҜгӮ’з”ЁгҒ„гӮӢгҖӮ$\mathrm{dp}[i][j]$гӮ’гҖҢгҖҢе…Ҳй ӯ$i$й …гҒ®й Ҷз•ӘгӮ’гҒқгӮҚгҒҲгҒҰгӮігӮ№гғҲгҒҢ$j$гҒЁгҒӘгӮӢгӮҲгҒҶгҒӘеӨүжӣҙж–№жі•гҒ§з”ҹжҲҗгҒ•гӮҢгӮӢгғҡгӮўеҲ—гҒ®жң«е°ҫгҒ®гғҡгӮўгҖҚгҒ®жңҖе°ҸгҖҚгҒЁгҒҷгӮӢгҖӮ йҒ·з§»гҒҜеӨүжӣҙ0, 1, 2еӣһгҒ®3йҖҡгӮҠгӮ’жӣёгҒ‘гҒ°гӮҲгҒ„гҖӮеӨүжӣҙ0гҒҜ$(x _ i, y _ i)$гҒҢгҒҷгҒ§гҒ«йҒ·з§»е…ғгӮҲгӮҠеӨ§гҒҚгҒ„е ҙеҗҲгҒ®гҒҝгӮ’иҖғгҒҲгӮӢгҖӮеӨүжӣҙ2гҒҜйҒ·з§»е…ғгҒЁеҗҢгҒҳгғҡгӮўгҒ«еӨүжӣҙгҒ—гҒҰгӮ„гӮҢгҒ°гӮҲгҒ„гҖӮеӨүжӣҙ1гҒҜгҒ“гӮҢгӮүгӮҲгӮҠе°‘гҒ—йӣЈгҒ—гҒ„гҒҢгҖҒгҖҢеҚҲеҫҢгҒ®жҲҗзёҫгӮ’йҒ·з§»е…ғгҒЁеҗҢгҒҳ or +1гҒ«гҒҷгӮӢгҖҚгҒЁгҖҢеҚҲеүҚгҒ®жҲҗзёҫгӮ’йҒ·з§»е…ғгҒЁеҗҢгҒҳ or 0гҒ«гҒҷгӮӢгҖҚгҒ®4йҖҡгӮҠгҒ®гҒҶгҒЎжңүеҠ№гҒӘгӮӮгҒ®гӮ’гҒҷгҒ№гҒҰе®ҹиЎҢгҒҷгӮҢгҒ°гӮҲгҒ•гҒқгҒҶгҖӮ

и§Јжі•гҒ®жҰӮиҰҒ

иІӘж¬Іжі•гғҷгғјгӮ№гҒ®гҖҢжңҖйҒ©гҒӘжұәгӮҒж–№гҖҚгҒҝгҒҹгҒ„гҒӘгӮӮгҒ®гҒҜгҒӘгҒҸгҒҰгҖҒе…ЁжҺўзҙўгӮ’гғҷгғјгӮ№гҒ«гҒҷгӮӢгҒ—гҒӢгҒӘгҒ„гҖӮз«ҜгҒӢгӮүзўәе®ҡгҒ•гҒӣгӮӢгҒЁгҒҷгӮӢгҒЁгҖҒдёҖз•ӘжңҖеҫҢгҒҢдҪ•гҒ гҒЈгҒҹгҒӢд»ҘеӨ–гҒ®жғ…е ұгӮ’жҚЁгҒҰгҒҰгӮҲгҒ„гҒҹгӮҒгҖҒdpгҒҢеҮәжқҘгҒқгҒҶгҖӮ

зҺҮзӣҙгҒ«гӮ„гӮӢгҒЁ$\mathrm{dp}[i][j][k]$гӮ’е…Ҳй ӯ$i$й …гӮ’еҮҰзҗҶгҒ—гҒҰжңҖеҫҢгҒҢ$(j, k)$гҒ«гҒӘгӮӢжңҖе°ҸгҒ®еӨүжӣҙеӣһж•°гҒЁгҒӘгӮҠгҒқгҒҶгҖӮгҒ“гӮҢгҒҜйҒ·з§»е…ҲгӮ’иүҜгҒ„ж„ҹгҒҳгҒ«жһқзӢ©гӮҠгҒ—гҒҰгӮӮ$\Omega(N ^ 3)$гҒЁгҒӢгҖӮ

$\mathrm{dp}[i][x][y] = \mathrm{dp}[i][p][q]$гҒ гҒЈгҒҹгҒЁгҒ—гҒҹгӮүгҖҒ$(x, y)$гҒЁ$(p, q)$гҒ®гҒҶгҒЎгӮҲгӮҠе°ҸгҒ•гҒ„ж–№гҒ гҒ‘дҝқжҢҒгҒ—гҒҰгҒҠгҒ‘гҒ°гӮҲгҒ„гҖӮпјҲгӮҲгӮҠеӨ§гҒҚгҒӘж–№гҒӢгӮүйҒ·з§»гҒ—гҒҰгӮӮжҳҺгӮүгҒӢгҒ«жңҖйҒ©и§ЈгҒ«гҒӘгӮҠгҒҲгҒӘгҒ„гҖӮпјү ж·»еӯ—гҒ®е…ҘгӮҢжӣҝгҒҲгғҶгӮҜгӮ’з”ЁгҒ„гҒҰгҖҒеҗҢгҒҳгӮігӮ№гғҲгӮ’е®ҹзҸҫгҒҷгӮӢгӮӮгҒ®гҒ®гҒҶгҒЎгҖҒжң«е°ҫгҒ®гғҡгӮўгҒҢжңҖе°ҸгҒ«гҒӘгӮӢгӮӮгҒ®гҒ гҒ‘гӮ’ж®ӢгҒҷгӮҲгҒҶгҒ«гҒҷгӮӢгҖӮ

$\mathrm{dp}[i][j]$гҒҜе…Ҳй ӯ$i$й …гӮ’еҮҰзҗҶгҒ—гҒҰгӮігӮ№гғҲгҒҢ$j$гҒЁгҒӘгӮӢзү©гҒ®гҒҶгҒЎгҖҒжңҖе°ҸгҒ®жң«е°ҫгғҡгӮўгҒЁгҒӘгӮӢгҖӮгҒ“гҒ®жҷӮзӮ№гҒ§$\Omega(N ^ 2)$гҖӮй…ҚгӮӢйҒ·з§»гӮ’иҖғеҜҹгҒҷгӮӢгҒЁ$O(1)$еҖӢгҒ—гҒӢгҒӘгҒ„гҒ®гҒ§гҖҒ$O(N ^ 2)$гҒЁгҒӘгӮӢгҖӮпјҲгӮҲгҒҸгҒӮгӮӢгӮӮгӮүгҒҶеҪўејҸгҒ«гҒ—гҒҰгғҮгғјгӮҝж§ӢйҖ гҒ§гҒӘгӮ“гҒЁгҒӢгҒҷгӮӢгҒҝгҒҹгҒ„гҒӘгҒ“гҒЁгҒҜз„ЎзҗҶгҒқгҒҶгҒӘгҒ®гҒ§з„ЎйӣЈгҒ«й…ҚгҒЈгҒҹж–№гҒҢгҒ„гҒ„гҖӮпјү

е®ҹиЈ…дҫӢ

ACжҸҗеҮә 

#include <iostream>
#include <vector>
#include <utility>

using namespace std;

constexpr int iINF = 1'000'000'000;

int cmp (const pair<int, int>& x, const pair<int, int>& y) {
    if (x.second == y.second) {
        if (x.first == y.first) return 0;
        return x.first < y.first ? -1 : 1;
    }
    return x.second < y.second ? -1 : 1;
}

int main () {
    while (true) {
        int n; cin >> n;
        if (n == 0) break;

        vector<int> x(n), y(n);
        for (int i = 0; i < n; i++) {
            cin >> x[i] >> y[i];
        }

        // dp[i][j] := жҳҮй ҶгҒ«iеҖӢеҮҰзҗҶгҒ—гҒҰгҖҒеӨүжӣҙjеӣһгӮ’гҒ—гҒҹгҒЁгҒҚгҖҒдҪңгӮҢгӮӢжңҖгӮӮе°ҸгҒ•гҒ„гӮ№гӮігӮў
        vector<pair<int, int>> dp(n + 1), ndp(n + 1);
        for (int i = 0; i < n + 1; i++) {
            dp[i] = make_pair(iINF, iINF);
        }
        dp[0] = make_pair(0, 0);

        for (int i = n - 1; 0 <= i; i--) {
            for (int j = 0; j < n + 1; j++) {
                ndp[j] = make_pair(iINF, iINF);
            }

            for (int j = 0; j < n + 1; j++) {
                if (dp[j] == make_pair(iINF, iINF)) continue;

                auto cur = make_pair(x[i], y[i]);
                // еӨүжӣҙ0еӣһ
                if (cmp(dp[j], cur) <= 0) {
                    if (cmp(cur, ndp[j]) <= 0) {
                        ndp[j] = cur;
                    }
                }

                if (j == n) continue;
                // еӨүжӣҙ1еӣһ
                // гҒ©гҒЎгӮүгӮ’еӨүжӣҙгҒҷгӮӢгҒӢгҒ§еҲҶгҒ‘гӮӢгҖӮ

                // еҚҲеүҚгҒ®йғЁ
                if (x[i] != 0
                        && cmp(dp[j], make_pair(0, y[i])) <= 0
                        && cmp(make_pair(0, y[i]), ndp[j + 1]) <= 0) {
                    ndp[j + 1] = make_pair(0, y[i]);
                }
                if (x[i] != dp[j].first
                        && cmp(dp[j], make_pair(dp[j].first, y[i])) <= 0
                        && cmp(make_pair(dp[j].first, y[i]), ndp[j + 1]) <= 0) {
                    ndp[j + 1] = make_pair(dp[j].first, y[i]);
                }

                // еҚҲеҫҢгҒ®йғЁ
                if (y[i] != dp[j].second
                        && cmp(dp[j], make_pair(x[i], dp[j].second)) <= 0
                        && cmp(make_pair(x[i], dp[j].second), ndp[j + 1]) <= 0) {
                    ndp[j + 1] = make_pair(x[i], dp[j].second);
                }
                if (y[i] != dp[j].second + 1
                        && dp[j].second + 1 < n
                        && cmp(dp[j], make_pair(x[i], dp[j].second + 1)) <= 0
                        && cmp(make_pair(x[i], dp[j].second + 1), ndp[j + 1]) <= 0) {
                    ndp[j + 1] = make_pair(x[i], dp[j].second + 1);
                }

                if (j == n - 1) continue;
                // еӨүжӣҙ2еӣһ
                // гҒ“гҒ®е ҙеҗҲгҒҜеүҚгҒЁгғ”гғғгӮҝгғӘеҗҲгӮҸгҒӣгӮҢгҒ°ok
                if (dp[j].first != x[i] && dp[j].second != y[i]) {
                    if (cmp(dp[j], ndp[j + 2]) <= 0) {
                        ndp[j + 2] = dp[j];
                    }
                }
            }

            swap(dp, ndp);
        }

        int ans = -1;
        for (int i = 0; i < n + 1; i++) {
            if (dp[i] != make_pair(iINF, iINF)) {
                ans = i;
                break;
            }
        }

        cout << ans << "\n";
    }
}
